Binding an SSH Launcher to a GNU Screen Hotkey

Two lines added to .screenrc: A launcher window: screen -t launcher /bin/sh -c 'while true; do echo -n "Hostname: "; read host; screen -t $host ssh $host; clear; done' A keybinding: bindkey "13" select 0 — maps Ctrl-K to switch to the launcher window. Pressing Ctrl-K and entering a hostname automatically creates a new GNU Screen window titled with that hostname and initiates SSH.

CFEngine Enterprise Tip: Showing Hosts with Broken RPM Databases

RPM database corruption is a common problem on Red Hat Linux at scale. Custom SQL query for CFEngine Enterprise to identify affected systems: select hosts.hostname,changetimestamp from promiseexecutions inner join hosts on promiseexecutions.hostkey = hosts.hostkey where logmessages::text ilike '%rpm%db3%' and changetimestamp > current_timestamp - '24 hours'::interval order by changetimestamp; Searches the past 24 hours of promise executions for log messages containing “rpm” and “db3”, returning hostnames and timestamps.

Dumping RPM Metadata

Needed to identify which RPM metadata field stores the “el6” value visible in package version output. One-liner to dump all available RPM metadata fields: rpm -q kernel-2.6.32-504.el6.x86_64 --queryformat "$(for f in `rpm --querytags`; do echo \n $f = %{${f}} ' ' ; done )" The “RELEASE” field contains the relevant “504.el6” value, along with derived fields like EVR, NEVR, NEVRA, and NVR.
