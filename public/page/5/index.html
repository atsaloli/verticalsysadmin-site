Give Your Sysadmin Career a Boost with the LOPSA Mentorship Program

LOPSA (League of Professional System Administrators) mentorship initiative — volunteer-driven, connecting experienced sysadmins with aspiring professionals. Mentors help “define the project goals, then let them work independently,” acting as coaches. Projects should have clear endpoints (e.g., setting up specific infrastructure or services). Proteges describe their desired projects in one paragraph, get matched with mentors, and submit brief project summaries upon completion. About LOPSA: independent nonprofit, Mount Laurel, NJ. Phone: 856-439-0500 or 800-285-2141. Mentorship can develop “professional relationships that can last a lifetime.”

Relative Origin of Cfengine, Puppet and Chef

Cfengine: Pioneered configuration description languages for Unix, introduced “classes” for conditional logic. Version 2 added convergence and self-healing. Puppet: Luke Kanies, a power Cfengine 2 user, created it to address his frustrations. “Graph-based and model-driven” with a “simple DSL language (designed to be simple, safe and human readable)” plus Ruby extensions. Chef: Adam Jacobs, dissatisfied with Puppet’s non-deterministic default execution ordering, built Chef with “tightly ordered” sequence control. Innovative cloud provisioning for VM instantiation and automated configuration. Originated in the Ruby-on-Rails cloud ecosystem. ...
