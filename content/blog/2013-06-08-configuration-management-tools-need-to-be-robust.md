title: "Configuration Management Tools Need to Be Robust: How NASA Builds Software"
date: 2013-06-08
author: "Aleksey Tsalolikhin"
---

Configuration management tools have become essential infrastructure components and must therefore be exceptionally reliable. References Dr. Holzmann's presentation on NASA/JPL software development practices.

The argument: as configuration management tools have become foundational to modern infrastructure operations, they deserve the same quality assurance and robustness engineering that aerospace organizations apply to mission-critical systems.
