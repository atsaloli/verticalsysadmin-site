Configuration Management tools have become a critical part of our infrastructure. We expect and depend on these tools being robust.

For the tool makers I offer [my notes on Dr. Holzmann's talk on how NASA/JPL builds robust software](http://www.verticalsysadmin.com/making_robust_software/ "notes"). The first part introduces and describes the challenge, the second part contains the solutions.
