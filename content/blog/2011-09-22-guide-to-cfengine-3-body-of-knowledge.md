## Getting Started

[CFEngine Quickstart](http://cfengine.com/manuals/cf3-quickstart.html), then the [CFEngine 3 Concept Guide](http://cfengine.com/manuals/cf3-conceptguide.html) covering system automation fundamentals, components, and knowledge management principles.

## Core Documentation

- [Glossary of Terms](http://www.cfengine.com/manuals/cf3-glossary.html)
- [Tutorial](http://www.cfengine.com/manuals/cf3-tutorial.html): system automation, components, bodies/bundles, initial promises, network services
- [CFEngine 3 Best Practices](http://www.cfengine.com/manuals/cf3-bestpractice.html): policy style, quality assurance
- [Troubleshooting Guide](http://www.cfengine.com/pages/troubleshoot): apparent hangs, memory concerns
- [CFEngine Solutions](http://www.cfengine.com/manuals/cf3-solutions.html): from basic (file copying) to advanced (PXE boot)
- [Reference Manual](http://www.cfengine.com/manuals/cf3-reference.html): auto-generated from source code
