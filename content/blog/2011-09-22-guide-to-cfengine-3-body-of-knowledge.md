Organized learning resources for CFEngine 3.

**Getting started:** CFEngine Quickstart, then "CFEngine 3 Concept Guide" (system automation fundamentals, components, knowledge management principles).

## Core Documentation

- Glossary of Terms
- Tutorial: system automation, components, bodies/bundles, initial promises, network services
- Best Practices Guide: policy style, quality assurance
- Troubleshooting Guide: apparent hangs, memory concerns
- Solutions Guide: from basic (file copying) to advanced (PXE boot configuration)
- Reference Manual: auto-generated from source code

## Learning Materials

Mark Burgess's four-part video series; Neil Watson's practical examples and cookbook.

## Policy Libraries

Community Open Promise-Body Library (COPBL), Modular Bundles, Orion Cloud Pack.
